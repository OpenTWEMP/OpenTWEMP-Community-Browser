[font="Georgia"][size="4"][b]Что такое OpenTWEMP Community Browser ?[/b][/size][/font]

Это универсальный клиент для повседневного использования многочисленных модов M2TW, установленных на Вашем ПК.
Название программы может показаться непонятным, поэтому здесь понадобится разъяснение:
[b]OpenTWEMP[/b] - программа является частью глобального проекта Open Total War Engine Modding Platform,
[b]Community[/b] - программа ориентирована на использование сообществом, т.е. геймерами и модмейкерами,
[b]Browser[/b] - программа по своей философии является обозревателем (браузером) для модификаций M2TW.
Универсальность программы проявляется главным образом в уважении Ваших индивидуальных пользовательских предпочтений.

[font="Georgia"][size="4"][b]Что представляет собой текущая реализация программы ?[/b][/size][/font]

OpenTWEMP Community Browser BETA 2025 - масштабный релиз, основанный на функциональности предыдущих Preview-версий и доведенный до ума.
На подготовку бета-версии ушло около года работы марш-бросками - в результате мы имеем на сегодня бета-релиз раннего публичного доступа.
В текущую реализацию клиента вошли только наиболее протестированные функции программы, которые на данный момент оказались готовы к выпуску.
В действительности заложен фундамент для гораздо большей функциональности, однако частично реализованные функции пока что заблокированы.
На сегодняшний день можно уверенно констатировать, что это лучшая версия программы и фактически полноценный универсальный лаунчерный движок.

[font="Georgia"][size="4"][b]Фундаментальные нововведения бета-версии[/b][/size][/font]

[font="Palatino Linotype"][size="3"][b]Полная поддержка опций игровой конфигурации M2TW[/b][/size][/font]
Реализована поддержка практически ВСЕХ опций конфигурации M2TW (согласно описанию игровой конфигурации из Docudemons v5.4).

[font="Palatino Linotype"][size="3"][b]Интегрированная персонализация модов M2TW с помощью распространяемых предустановок[/b][/size][/font]
Реализован новый упрощенный способ персонализация для Ваших модов M2TW через механизм распространяемых предустановок.

[font="Palatino Linotype"][size="3"][b]Настраиваемая персонализация модов M2TW с помощью пользовательских предустановок[/b][/size][/font]
Реализована возможность самостоятельно персонализировать Ваши моды M2TW через механизм пользовательских предустановок.

[font="Palatino Linotype"][size="3"][b]Воспроизведение фонового музыкального сопровождения для Ваших модов с помощью предустановок[/b][/size][/font]
Реализованы базовые функции воспроизведения фонового музыкального сопровождения Ваших модов в клиенте на основе предустановок.

[font="Palatino Linotype"][size="3"][b]Поддержка встроенной персонализации для популярных модов M2TW[/b][/size][/font]
Обеспечена поддержка встроенной персонализации наиболее популярных модов M2TW через механизм распространяемых предустановок.

[font="Georgia"][size="4"][b]Системные требования[/b][/size][/font]

Требуется наличие установленной M2TW на Вашем компьютере (современная Steam-версия или классическая дисковая версия).
Ожидается, что Ваш ПК находится под управлением 64-битной версии ОС Windows 7/8/10/11 (есть поддержка [url="https://dotnet.microsoft.com/en-us/download/dotnet/8.0"].NET Desktop Runtime 8.0[/url]).
[b]Примечание:[/b] Вышеупомянутый .NET Desktop Runtime уже входит в автономный исполняемый файл программы.

[font="Georgia"][size="4"][b]Когда ждать выпуск следующей версии OpenTWEMP Community Browser ?[/b][/size][/font]

В настоящий момент затрудняюсь сообщать о конкретных планах по дальнейшему развитию проекта в ближайшей перспективе.

Последние 2 года работы марш-бросками над данной программой заставляют всерьёз задуматься о смене подходов к планированию дальнейших релизов.
Масштаб кодовой базы программы с момента выпуска версии Preview 2023.1 серьезно вырос - чем дальше, тем тяжелее работать рейдерскими методами.
Есть большая вероятность, что в дальнейшем я перейду на некоторый итерационный подход с более точечными компактными релизами новых версий.

Мои конкретные планы на ближайшие несколько месяцев - взять творческий перерыв, чтобы отдохнуть от проекта и накопить новый заряд энтузиазма.
